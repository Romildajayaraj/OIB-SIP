🌙 Night Companion – A Telegram Bot for Women’s Safety

🔹 Overview

Night Companion is a Telegram bot designed for women’s safety, offering real-time support, location tracking, and emergency assistance. It ensures that women feel safer while traveling alone at night.

🔹 Problem Statement

Women’s safety during late-night travel is still a major concern. Many safety apps are complex or require multiple steps in emergencies. This project addresses the gap by offering an instant-access solution directly on Telegram.

🔹 Key Features

  • 📍 Live Location Sharing
  • 🚨 SOS Alerts with Location
  • 🤖 Chatbot Assistance for emergency helplines
  • 🕒 Check-in System to confirm user safety
  • 🔒 Privacy-focused with minimal data

🔹 Tech Stack

🐍 Python
📲 Telegram Bot API
☁️ Hosting: Heroku / PythonAnywhere
🔧 Libraries: python-telegram-bot, requests, geopy

🔹 My Role

I designed the bot’s workflow, implemented the SOS alert & location-sharing feature in Python, tested with real scenarios, and ensured the bot was easy to use inside Telegram.

🔹 Outcome & Impact

The bot was successfully developed and tested, improving safety confidence for women at night. It enhanced my skills in Python, APIs, and real-world problem-solving.
